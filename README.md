# PSD Tool for AE

After Effects 2026向けの、PSDレイヤー表示制御用拡張機能です。

読み込んだPSDファイルのレイヤー構造をツリー表示し、チェックボックスやラジオボタンで表示/非表示を切り替え、その状態をコンポジションのキーフレームとして即座に反映します。

簡易的な口パク・目パチのアニメーション制作や、差分管理に役立ちます。

このほか、キーフレームを一括操作する便利な機能と、音声合成ソフトウェア等で出力したファイルを読み込むことで、テキストレイヤーに字幕を自動生成するスクリプトも付属しています。

## PSDファイルの仕様
基本的には[立ち絵支援ツール PSDTool様にて解説されている仕様](https://bl.oov.ch/2016/08/psdtool.html)に準拠しています。

フォルダ・レイヤー構造をそのままAfter Effectsのコンポジション階層にマッピングします。

フォルダ・レイヤー名の先頭に特定の記号を付与することで、表示切替の動作を変更できます。
- `*` (アスタリスク): ラジオボタンとして動作。同一階層内で排他的に選択されます。
- `!` (感嘆符): ロック (常時表示) として動作。常に表示され、操作できません。

## 機能

- **テーマ同期**: After EffectsのUIの明るさ設定に合わせて、パネルの色が自動的に変わります。
- **階層ツリー表示**: PSDのフォルダ・レイヤー構造を子孫まで含めて再帰的にツリー表示します。
- **ツリーの一括展開/折り畳み**:
    - `Alt` キーを押しながら `▶` / `▼` をクリックすると、配下のフォルダを一括で展開/折り畳みできます。
    - 画面上部の「すべてのフォルダを畳む」ボタンで全フォルダを折り畳むことができます。
- **表示切替**
    - **チェックボックス**: 通常のレイヤー。
    - **ラジオボタン**: レイヤー名またはフォルダ名の先頭が `*` の場合、同一階層内で排他的に選択されます。
    - **ロック (常時表示)**: レイヤー名またはフォルダ名の先頭が `!` の場合、常に表示されます（操作不可）。
- **キーフレーム操作**
    - **取得**: 現在位置のキーフレームを子孫レイヤーも含めて全て取得し記録します。
    - **移動**: 記録したキーフレームを任意の場所へ移動させることができます。
    - **一括シフト機能 (Shift+クリック)**: `Shift`キーを押しながら「取得」「削除」操作を行うと、現在位置から末尾までのキーフレームを一括で対象とします。また、`Shift`キーを押しながら「移動」を行うと、取得したキーフレーム同士の間隔を維持したまま全体をシフトさせることができます。
- **プリセット**
    - **保存/呼び出し**: レイヤーの表示状態をプリセットとして保存・呼び出しできます。
    - **インポート/エクスポート**: プリセットをJSON形式で保存・読み込みできます。
- **字幕作成**: テキストレイヤーに字幕を自動生成できます。字幕の内容は音声合成ソフトウェア等で出力したファイルを読み込むことができます。

## 使い方
### PSDファイルの読み込みと表示切替
1. PSDファイルをAfter Effectsのプロジェクトパネルにドラッグ＆ドロップして読み込みます。必ず「**コンポジションとして読み込み**」を選択してください。

2. メニューバーの **ウィンドウ > 拡張機能 > PSD Tool AE** を選択してパネルを開きます。

3. 「再読み込み」ボタンをクリックして、PSDファイルの内容をパネルに反映させます。

4. ツリー上のチェックボックスやラジオボタンを操作して、レイヤーの表示/非表示を切り替えます。変更は即座にコンポジションに反映されます。

5. 「キーフレーム操作」欄の「取得」ボタンをクリックして、現在位置のキーフレームを子孫レイヤーも含めて全て取得します。さらに一括シフト機能を利用する場合は、`Shift`キーを押しながらクリックすると以降のキーフレームも一括で取得します。

6. 「キーフレーム操作」欄の「移動」ボタンをクリックして、記録したキーフレームを任意の場所へ移動させることができます。複数キーフレームを取得している場合、`Shift`キーを押しながら「移動」をクリックすると、キーフレーム間の間隔を保持したまま一括シフト移動します。

7. 表示状態のプリセットを保存したい場合は、「Presets」欄の「+」ボタンをクリックします。プリセット名を入力するダイアログが表示されるので、任意の名前を入力します。保存したプリセットはリストから選択して呼び出すことができます。

8. プリセットのインポート/エクスポートは、「Presets」欄の「In」/「Ex」ボタンから行えます。エクスポートされたJSONファイルは、他のプロジェクトと共有することができます。

### 字幕の自動生成
1. 音声合成ソフトウェア等で出力した音声データと、テキストファイルを用意します。
    - お使いの音声合成ソフトウェアにて、音声出力時に「台詞をファイルに保存」などのオプションを有効化してください。
    - ファイル名は同一にして、拡張子だけが異なる必要があります。
	  - 例: `000-AAAA-BBBB-CCCC.wav` と `000-AAAA-BBBB-CCCC.txt`
    - ファイル名には、字幕を割り当てたいキャラクター名に対応する文字列が含まれている必要があります。どの部分をキャラクター名として扱うかは正規表現で指定できます。正規表現内の丸括弧`()`で囲んだ「キャプチャグループ」が該当文字列として抽出されます。お使いの音声合成ソフトウェアのファイル命名規則に合わせて、正規表現を適宜調整してください。
	  - 例: デフォルトの `.*-.*-(.*)-.*` は `000-AAAA-音街ウナ-000.txt` から `音街ウナ` を抽出します。
    - 抽出した文字列は、字幕レイヤー名と前方一致で照合されます。
	  - 例: 抽出結果が `音街ウナ` の場合、`音街ウナ` や `音街ウナ字幕` といったレイヤー名に字幕が生成されます。
    - テキストファイルの内容は、音声ファイルに対応する字幕のテキストです。このテキストの内容がそのまま字幕レイヤーのテキストプロパティに反映されます。

2. 音声ファイルをAfter Effectsのプロジェクトパネルにドラッグ＆ドロップして読み込み、タイムラインに配置します。

3. メニューバーの **ウィンドウ > ImportSubtitles.jsx** を選択してパネルを開きます。

4. 字幕を反映させたい音声レイヤーを選択し、「テキストを反映」ボタンをクリックします。

## 動作環境

- Adobe After Effects 2026 (またはそれ以降のCEP 9.0対応バージョン)
- Windows / macOS

## インストール方法 (開発者モード / 手動インストール)

ZXPインストーラーを使用しない、未署名拡張機能としてのインストール手順です。

### 1. デバッグモードの有効化
未署名の拡張機能を読み込むために、レジストリ(Windows)またはターミナル(macOS)の設定が必要です。

#### Windows (PowerShell)
以下のコマンドをPowerShellで実行し、Adobe CEPのデバッグモードを有効にします。
```powershell
$versions = 10..20
foreach ($v in $versions) {
    reg add "HKCU\Software\Adobe\CSXS.$v" /v PlayerDebugMode /t REG_SZ /d 1 /f
}
```

#### macOS (Terminal)
```bash
defaults write com.adobe.CSXS.10 PlayerDebugMode 1
defaults write com.adobe.CSXS.11 PlayerDebugMode 1
# ...必要に応じてバージョン番号を変えて実行
```

### 2. 拡張機能フォルダへの配置

本リポジトリのデータを、AdobeのExtensionフォルダに配置します。

#### Windows

##### 字幕作成スクリプトファイル
`C:\Program Files\Adobe\Adobe After Effects 2026\Support Files\Scripts\ScriptUI Panels`に、`リポジトリの中身/jsx/ImportSubtitles.jsx`をコピーしてください。

##### PSD Tool AEの拡張機能ファイル
`C:\Users\<ユーザー名>\AppData\Roaming\Adobe\CEP\extensions\`  
上記のフォルダの中に `psdtool_ae` という名前でフォルダを作成し、リポジトリの中身をすべてコピーしてください。

#### macOS
※製作者はWindowsユーザーのため、macOSでの動作確認は行っていません。以下はAIによる出力であり、正確性は保証できません。適宜読み替えてください。

##### 字幕作成スクリプトファイル
`/Applications/Adobe After Effects 2026/Scripts/ScriptUI Panels`に、`リポジトリの中身/jsx/ImportSubtitles.jsx`をコピーしてください。

##### PSD Tool AEの拡張機能ファイル
`/Library/Application Support/Adobe/CEP/extensions/`  
または  
`~/Library/Application Support/Adobe/CEP/extensions/`

ダウンロードしたフォルダを上記のフォルダの中にコピーしてください。

### 3. After Effectsの起動
After Effectsを起動し、メニューバーの **ウィンドウ > 拡張機能 > PSD Tool AE** を選択してパネルを開きます。
